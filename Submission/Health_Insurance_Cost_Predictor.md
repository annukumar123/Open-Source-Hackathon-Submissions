## Project Details

### Project Title
Health Insurance Cost Predictor

### Project Description
The Health Insurance Cost Predictor is a fully deployed, end-to-end machine learning application designed to automate the estimation of annual medical insurance premiums. Developed during an engineering internship, the system leverages the Random Forest Regression algorithm to analyze historical medical expense datasets, effectively identifying and quantifying how crucial risk variables like smoking status, BMI, and age dictate overall insurance pricing.

To bridge the gap between model training and real-world execution, the project features a custom-built, low-latency API architecture serving predictive outputs instantly. The frontend provides a clean, user-centric dashboard where individuals can input their health demographics and receive verified financial forecasts in real time. 

### Tech Stack Used
- **Machine Learning & Data:** Python, Scikit-learn, Random Forest Regressor, Pandas, NumPy
- **API Framework:** FastAPI (Asynchronous Python Web Framework)
- **Deployment & Cloud Infrastructure:** Render Cloud Platform
